Built for beauty
and speed

Cross-platform LGPL-licensed desktop GUI framework for C and Rust, backed by the Mozilla WebRender rendering engine.

  • Memory: 23MB
  • CPU: 0%

The UI structure is created via composition instead of inheritance. Callbacks can modify the application data and then tell the framework to reconstruct the entire UI again if necessary - not on every frame.
