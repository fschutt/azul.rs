Cross-platform MIT-licensed desktop GUI framework for C and Rust, backed by the Mozilla WebRender rendering engine.

Azul can render infinitely large datasets (such as a table, shown here) while using a comparably small amount of memory.
Users can specify special IFrameCallbacks that - given the size and scroll position of the DOM node - only return the currently visible cells. All cells can share their CSS style efficiently via pointers, so that properties do not get duplicated in memory.
